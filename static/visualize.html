Vector Space Visualization

Understanding Vector Embeddings

This visualization shows how your documents are positioned in the vector space. Documents with similar content appear closer together in this space.

When you search, your query is converted to a vector and the system finds the document vectors that are closest to it.

Document Collection Overview

This system uses vector embeddings to represent documents and compare their similarity. Each document is converted to a high-dimensional vector (768 dimensions) that represents its semantic meaning.

When searching, your query is converted to a vector and compared against all document vectors to find the most similar matches.

Uploaded Documents

You can view information about your uploaded documents with:

python file_browser.py

And search documents from the command line with:

python file_browser.py --search "your search query"
